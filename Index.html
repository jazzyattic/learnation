Google Chrome is a cross-platform web browser developed by Google.
It was first released in 2008 for Microsoft Windows, built with free software components from Apple WebKit and Mozilla Firefox.
Versions were later released for Linux, macOS, iOS, and also for Android, where it is the default browser.



Microsoft Edge is the default web browser on Windows 10,
Windows 10 Mobile, Windows 11, Xbox One, and Xbox Series X and Series S consoles,
replacing Internet Explorer 11 and Internet Explorer Mobile.



Firefox is a free,
open source web browser developed by the Mozilla Foundation and Mozilla Corporation in 2004.
The Firefox web browser can be used with Windows, Mac and Linux operating systems, as well as Android and iOS mobile devices.



Opera is a multi-platform web browser developed by its namesake company Opera.
The browser is based on Chromium, but distinguishes itself from other Chromium-based browsers through
its user interface and other features.



Safari is a web browser developed by Apple.
It is built into Apple's operating systems, including macOS, iOS, and iPadOS,
and uses Apple's open-source browser engine WebKit,
which was derived from KHTML.
